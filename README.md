Projet flutter
Description :

Application Flutter permettant de créer et gérer des voyages avec des activités.

Fonctionnalités :
  - Rechercher une ville
  - Créer un voyage
  - Choisir une date
  - Ajouter des activités
  - Marquer une activité comme terminée
  - Voir la météo
  - Voir une activité sur la carte
