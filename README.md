Projet flutter
Description :

Application Flutter permettant de créer et gérer des voyages avec des activités.

Fonctionnalités :
  - Rechercher une ville
  - Créer un voyage
  - Choisir une date
  - Ajouter des activités
  - Marquer une activité comme terminée
  - Voir la météo
  - Voir une activité sur la carte


# Démarrage du projet

## 1️) Démarrer le serveur backend
  Ouvrir un terminal, se placcer dans le dossier "backend" et exécuter :

```bash
npm start
```
Cette commande doit s'effectuer dans le dossier "backend".
Celle-ci a pour but de lancer le serveur MongoDB.

## 2️) Démarrer l'application Flutter

Ouvrir un autre terminal, se placer dans le dossier "flutter" et exécuter :
```bash
flutter run
```
L'application se lancera sur votre émulateur connecté.
